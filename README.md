# Fullstack Account

A fullstack account management project using pnpm workspace.

## Project Structure

```
.
├── apps/          # Applications
├── packages/      # Shared packages
└── pnpm-workspace.yaml
```

## Getting Started

### Install Dependencies

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

## Workspace Packages

This project uses pnpm workspace to manage multiple packages and applications.
